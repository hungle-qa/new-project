# Interactive Table Builder

A visual table builder with cell merging, styling controls, and component insertion capabilities.

## Features

- **Grid Creation**: Default 5x5 editable table
- **Cell Selection**: Click to select, Ctrl+click for multi-select
- **Cell Merging**: Merge adjacent cells, unmerge to restore
- **Dynamic Grid**: Add rows and columns dynamically
- **Styling Controls**: Border color and width customization
- **Component Library**: Searchable sidebar with insertable components
- **Export**: JSON export of table structure
- **Reset**: Clear all changes and restore default grid

## Pages

- `pages/builder.html` - Main builder interface

## Design System Components Used

- ButtonWhite - Action buttons in header
- LeftSidebar - Component library sidebar
